# 🧙‍♂️ ImageMagick Command Builder

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A powerful GUI for visually building complex [ImageMagick](https://imagemagick.org/) command-line strings. This tool acts as a "co-pilot," helping you discover and apply over 100 different options without having to memorize them.

![Screenshot of the Magick Builder UI](https://i.imgur.com/your-screenshot-url.png) <!-- Replace with a real screenshot URL -->

---

## ✨ Features

*   **Comprehensive UI:** Access over 100 of ImageMagick's most common flags, organized into logical categories.
*   **Live Command Preview:** See the command-line string build in real-time as you toggle options and adjust values.
*   **Instant Image Preview:** Load an image to see a preview. Supports standard formats (`JPG`, `PNG`, `GIF`) and even `TIFF` files.
*   **Flexible Loading:**
    *   Load images from your local disk.
    *   Paste an image URL to fetch it directly.
    *   Drag and drop files and folders.
*   **Batch Mode:** Automatically detects when a folder or multiple files are dropped, switching to a batch-friendly command structure (`*.jpg`, `converted-%d.png`).
*   **Responsive Design:** A unique cyberpunk-themed interface that works on both desktop and mobile.
*   **Helper Utilities:**
    *   **Clear All:** Instantly reset all selected options to their default state.
    *   **Built-in Help:** A comprehensive help menu with installation instructions and a full command reference.

## 🚀 Getting Started

This is a pure client-side application. However, for the best experience and to avoid browser security limitations, it's recommended to run it via a local web server.

### Recommended Method: Local Web Server

This method ensures all features, like dropping folders with spaces in their names, work correctly in all browsers.

1.  Make sure you have Python installed.
2.  Open a terminal in the project's root directory.
3.  Run the following command:
    ```bash
    python -m http.server
    ```
4.  Open your web browser and navigate to `http://localhost:8000`.

### Alternative Method: Direct File Access

You can also open the `index.html` file directly in your browser.

> **Note:** When using this method, some browsers (like Chrome and Brave) may have issues reading dropped folders if their names contain spaces or special characters. Firefox is generally more lenient.

## ⚙️ Core Workflow

This tool is designed to be a **command builder**, not an in-browser converter. The workflow is simple:

1.  **Load an Image:** Load a sample image to get a feel for the effects.
2.  **Build Your Command:** Use the UI to select and configure the desired ImageMagick flags.
3.  **Copy the Command:** Click the "Copy CLI" button to copy the generated command string to your clipboard.
4.  **Execute Locally:** Paste the command into your own terminal and run it. This gives you the full, native speed and power of your local ImageMagick installation.

## 📦 Installation

To use the commands generated by this tool, you must have ImageMagick installed on your system.

The application's help menu (`?` icon) contains detailed instructions, including a recommended one-line PowerShell command for easy installation on Windows.

```powershell
# Example Windows installation command from the app's help menu
$html = (New-Object System.Net.WebClient).DownloadString('https://imagemagick.org/archive/binaries/'); $exe = [regex]::Match($html, 'ImageMagick-7[\d\.\-]+Q16-HDRI-x64-dll\.exe').Value; $path = "$env:TEMP\magick.exe"; (New-Object System.Net.WebClient).DownloadFile("https://imagemagick.org/archive/binaries/$exe", $path); Write-Host "Downloading and installing silently... please wait." -ForegroundColor Yellow; Start-Process -FilePath $path -ArgumentList "/VERYSILENT /SUPPRESSMSGBOXES /NORESTART /SP-" -Wait -NoNewWindow; Write-Host "ImageMagick Installed Successfully!" -ForegroundColor Green
```

> **Important:** After installing ImageMagick, you must restart any open terminals for the `magick` command to be recognized.
